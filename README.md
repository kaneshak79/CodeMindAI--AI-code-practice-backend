# 🧠 CodeMindAI Backend

AI-powered coding assessment backend built with **FastAPI**, **Groq LLM**, and **MongoDB**.

It generates coding questions and evaluates user code using AI-based test simulation.

---

# 🚀 Features

- AI-powered coding question generation (LeetCode-style)
- AI-based code evaluation
- Automatic test case generation (AI simulated)
- Pass / Fail analysis of submissions
- Stores questions & submissions in MongoDB
- REST API for frontend integration

---

# 🏗️ Tech Stack

- FastAPI
- Python 3.10+
- Groq LLM (LLaMA 3.3 70B)
- MongoDB
- python-dotenv
